# 4팀- 러쉬코리아 (Beauty Trend Style)

## 팀원

### Front-end

- 강루비, 김대호, 양석문, 주혜린

### Back-end

- 김정훈, 정다영, 정승렬

## 사용한 기술

### Front-end

- React, Sass

### Back-end

- Node.js, MySQL, Express

### co-work

- git, slack, trello, notion
